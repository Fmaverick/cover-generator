「小红书图文封面生成系统 (Pro)」
一款基于 React 开发的极简图文生成工具。本工具专为内容创作者设计，支持自动分页、多种模版切换以及批量导出功能，能够帮助用户快速生成具有高质感的小红书封面与内页。

「核心功能」
- 多模版支持：内置 INS 风语录、iOS 备忘录、古籍书页、纯净笔记等多种视觉风格。

- 智能自动分页：输入长文本后系统会自动根据容器高度进行切分，生成多张排版统一的图片。

- 高度自定义能力：支持实时调整背景色彩、文字颜色、高亮标注色以及整体字号缩放。

- 一键批量导出：支持将所有生成的页面一键打包为 ZIP 压缩包，方便快速发布。

- 隐私安全保障：全本地浏览器端渲染，无需上传数据至服务器，确保内容私密性。

「使用指南」
- 内容输入：在左侧控制面板输入封面标题以及需要展示的长文本内容。

- 风格选择：在模版区域点击切换不同的视觉风格，系统将自动应用对应的配色方案。

- 细节微调：利用底部的取色器调整背景或文字细节，确保视觉效果符合预期。

- 保存作品：预览确认无误后，点击“导出全套”按钮即可获取高清图片压缩包。

「技术栈」
- 框架：React 18

- 样式：Tailwind CSS

- 绘图：Canvas API

- 导出：JSZip & FileSaver.js
